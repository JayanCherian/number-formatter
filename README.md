Number Formatter
================

A small library that adds commas to numbers

## Installation

    npm install @jayancherian/number-formatter

## Usage

    var numFormatter = require('@jayancherian/number-formatter');
    var formatted = numFormatter(10000)

    Output should be 10,000

# Tests

    npm test
